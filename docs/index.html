9) We would like to point out the following limitations:

  • We did not implement “recovery from a power outage”.
  • We did not implement checks for the container name, so a naïve user might type in “NAN” as a container name and we would accept it.
  • We did not implement duplicate name substitution. So, if there are duplicate named containers, and a user requests one of them, we will retrieve the one she asked for literally, even if one of the duplicates would have been cheaper to retrieve.
  • If there are more than 30 containers to be offloaded, our program may hang or crash (but this never happens for less than 30 containers)
